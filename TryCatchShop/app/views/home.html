Try Catch Shop Example

This is a simple app to simulate basic funcionality of an e-commerce. in the Admin section (you need to be logged in as Admin) you can add Products with picture. in shop area you can buy them. To log in click on the Log in button and on register if you don't have an account

In this example I've use the Asp.Net MVC5 WebaPI 2 version with Asp.Net.Identity.Owin and token oauth authentication.

Hosted on
Microsft Azure on cloud!
